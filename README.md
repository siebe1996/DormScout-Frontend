# DormScout Frontend

This is a school project made for Mobile Application Development for Odisee Ghent

## .Env

you'll need to create a .env file with this in

```
EXPO_PUBLIC_MAPS_API_KEY = your_maps_api_key
```

## Installation

```

git branch -M main
git push -uf origin main
npm install
```
